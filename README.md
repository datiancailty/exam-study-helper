# 刷题学习助手

已接入两个 Word 题库的考试练习原型：

- **GE机组培训题库**：单选、多选、简答
- **自控专业试题_0228**：单选、多选
- 题库和题型可通过页面顶部筛选
- 答案已从题目显示中分离，提交后才显示
- 单选/多选按标准答案精确判定；答对自动下一题，答错停留学习
- 简答题提交后展示 Word 中的参考答案，供自我对照，不做机械文本判分

题目来源于用户提供的 Word 文档。Word 中少数排版或答案标记不完整的题目已保留，并按“题库中未提供额外解析”处理，后续可以根据你的复核继续修正。

## 本地预览

```bash
python3 -m http.server 8765
```

然后访问 `http://localhost:8765`。
